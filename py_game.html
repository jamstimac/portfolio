Python Text Based Game

The premise of this game follows a dungeon crawler where you “level up” by collection items. The throne room situated to the West of many of the other rooms ends the game unless the player collects all items as they have not “leveled up” fully. Below is a video of gameplay including restarts, deadends, and the win condition. as well as the original code.

This code was made within an IDE for Python, and it utilizes the dictionary feature of the language. Most of the displayed text, commands, and error messages are stored within dictionaries under differing keys and called when needed through a decently long if-else tree. To update the game, the dictionaries and functions related would be moved to a different class file. This would also allow the creation of a full main function. Using an async def main() would also allow the addition of graphics and the porting of the game to the web.

def print_border(character='*', num_spaces=15):
      """ Prints border with specified character and number of repeats """
